# Jupyter Git Practice

Learning data science with Jupyter notebooks and Git/GitHub.

## Setup
1. Clone this repo
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate`
4. Install: `pip install -r requirements.txt`

## Notebooks
- `01_data_loading.ipynb` - Loading data from various sources
- `02_data_cleaning.ipynb` - Data cleaning techniques
- `03_exploratory_analysis.ipynb` - EDA with pandas
- `04_visualization.ipynb` - Creating plots
- `05_statistical_analysis.ipynb` - Statistical tests

## Git Practice
- Created branches for each analysis
- Practiced commits, merges, and pull requests
- Used nbdime for better notebook diffs
